# Crossref Metadata Search

A search UI for the [Crossref REST API](https://api.crossref.org/), built with Vue 3, TypeScript, and Vite.

## Getting Started

```sh
npm install
npm run dev
```

## Production Build

```sh
npm run build
npm run preview
```

## Testing

```sh
npm run test          # unit tests (Vitest)
npm run test:watch    # unit tests in watch mode
npm run test:e2e      # end-to-end tests (Playwright)
```

## Lint

```sh
npm run lint
```

## Type Check

```sh
npm run type-check
```

## Stack

- Vue 3 (Composition API, `<script setup>`)
- TypeScript
- Vite 7
- Vue Router 4
- SCSS with BEM
- Vitest + Vue Test Utils
- Playwright
- ESLint
